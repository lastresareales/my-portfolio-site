## рџЋ“ Customization

### Update Your Information
Edit `src/App.jsx` to change:
- Name
- Title/Occupation
- Email
- Social media handles

### Add Blog Posts
Edit `src/Components/Blog.jsx` and add to the `blogPosts` array.

### Update Portfolio Projects
Edit `src/Components/Portfolio.jsx` and add to the `projectList` array.

### Change Colors
Edit `src/styles.css` and update the CSS variables:
```css
:root {
  --primary-color: #4E567E;
  --secondary-color: #D2F1E4;
}
```
